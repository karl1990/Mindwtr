# Contributing

## Setup

```bash
bun install
```

## Desktop

```bash
bun desktop:dev
```

## Mobile

```bash
bun mobile:start
```

## Tests

```bash
bun run --filter mindwtr test
bun run --filter @mindwtr/core test
```

## Style

- TypeScript-first.
- Keep imports grouped (external/workspace/relative).
- Prefer accessibility-friendly queries in tests.

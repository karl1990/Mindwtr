# Mindwtr 0.4.6

## Highlights
- Added Spanish, German, and Japanese translations with broader coverage.
- Mobile app now supports landscape orientation on tablets.
- macOS releases now include both Apple Silicon and Intel builds.

## Desktop
- Settings language labels now translate correctly beyond Chinese.
- Release workflow builds both `macos-arm64` and `macos-x64` artifacts.

## Mobile
- Landscape rotation enabled (`orientation: default`).
- Expo Go gracefully falls back to JSON storage and disables notifications.
- Added missing SF symbol mapping for trash icon.

## Core
- Expanded translation coverage for Settings + common UI strings.
